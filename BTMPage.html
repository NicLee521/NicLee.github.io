Beat the Machine

About the Project

Beat the Machine is my most recent project which I am working on currently with a team of 9 other students. It is a top down sci-fi style twin-stick shooter with 3 serperate skill trees, loads of different enemy types, multiple different bosses, and many different powerups. The game is very similar to a sort of bullet hell/twin-stick shooter crossover due to the difficulty and amount of enemies on screen at once. In this project my main tasks so far have been the creation of the player controller (usable for both keyboard and controller), the skill tree framework, and all of the skill tree abilities.

Player Controller

The Player Controller is what will allow the player to use many different types of inputs from Xbox controllers, to keyboard and mouse, to Playstation controllers, and lastly using Nintendo Switch controllers. At the time of writing this it can handle both Keyboard and Mouse as well as Xbox Controllers.

Skill Tree Framework

The Skill Tree Framework is described in the name being that it is a framework for how the skill tree will work once its fully implemented. The player cna not choose any abilities in any tiers that aren't directly after the one the are currently in. For example if a player was specced into the 3rd tier on the vitality tree they are not able to go into the 4th tier on the utility tree but they can go into the 4th on the vitality tree.

Skill Tree Abilities

With there being 39 skills total in Beat the Machine, I've opted to not show all the ones I've done as of yet but just a few of the most recent ones. From top to bottom the ablilites are Rapid Fury, OverClock, Last Chance, and Wave Dash. These are just a few of the abilities featured in the Utility Tree.
